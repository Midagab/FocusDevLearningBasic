# My To-Do List App

A simple, clean, and intuitive task management app designed to help you stay on top of your daily responsibilities. Add tasks quickly, mark them as complete with a single click, and delete done items effortlessly.

## Features

- ➕ **Add Tasks** — Add new tasks with ease
- ✅ **Complete Tasks** — Mark tasks as complete using checkboxes
- 🗑️ **Delete Tasks** — Remove completed or unwanted tasks
- 📊 **Track Progress** — Monitor your progress with Active, Completed, and Total counters

## Getting Started

1. Open `index.html` in your web browser
2. Start adding your tasks!
3. Your tasks are saved automatically in your browser

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
